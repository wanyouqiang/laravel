# Laravel 路由
### Closure(闭包形式)
    `
        Route::get('/foo', function () {
            return 'laravel';
        });
    `

### GET/POST
    `
        Route::get('/foo', function () {
            return 'get';
        });
        Route::post('/post', function () {
            return 'post';
        });
        Route::any('/foo', function () {
            return 'get/post';
        });
        Route::match(['get', 'post'], '/foo', function () {
            return 'get/post';
        });
    `

### 直接渲染模板
    `
        Route::view('/foo', 'form'); // 找Resources目录下views下的form.blade.php
    `

### 命名路由
    `
        Route::get('foo', function () {
            return 'laravel';
        })->name('foo');

        Route::get('test', function () {
            return view('foo', ['id' => '2']);
        });
    `
